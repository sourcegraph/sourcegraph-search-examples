# AGENTS.md

## Commands
- `npm install` — Install dependencies
- `npm run dev` — Start dev server
- `npm run build` — Build examples.json and production assets
- `npm run build-examples` — Validate YAML and rebuild static/examples.json
- `npm test` — Run all tests with Vitest
- `npx vitest run tests/validate-examples.test.js -t "example links"` — Run single test by name

## Architecture
SvelteKit 2 + Svelte 5 single-page app (static adapter) displaying Sourcegraph search examples.
- `searchExamples.yaml` — Source of truth for examples (validated against `searchExamplesSchema.json`)
- `scripts/buildSearchExamples.js` — Builds `static/examples.json` from YAML
- `src/lib/` — Svelte components: SearchExamples, Filters, Results, Result
- `src/routes/` — SvelteKit routes (+page.svelte, +layout.svelte)
- `tests/` — Vitest tests validating schema, URLs, and required tags
- `.github/workflows/deploy.yml` — GitHub Actions workflow to build and deploy to Pages on push to main
- `docs/` — Build output directory (gitignored, generated by CI)

## Code Style
- ES modules (`"type": "module"`)
- Svelte 5 runes syntax; Tailwind CSS for styling
- Tests use Vitest with `test()`/`expect()` from `vitest`
- Examples require `pattern_type:` and `use_case:` tags
